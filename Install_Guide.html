Obsidian Theme - Installation Guide

Thank you for buying the Obsidian theme. Once you are happy with the theme and the service, please consider leaving a review. If you need help at any point, join the Discord for support: https://discord.gg/hkrxptUjmV

Recommended: Make a backup before installing. This install involves replacing theme files and a core class file. If anything goes wrong, a backup makes it easy to restore.
Important: Follow the steps in order. If something does not show up immediately, clear caches and refresh the admin panel.
Two parts: Theme install + Extensions install

Before you start: Backup (recommended)

Before making changes, it is strongly recommended to back up both your files and your database. At minimum, back up these locations:

  • /var/www/paymenter/themes/
  • /var/www/paymenter/extensions/
  • /var/www/paymenter/app/Classes/ (especially FilamentInput.php)

If you have a server control panel or snapshot feature, use that as well.

Section 1 - Install the Theme

1) Upload the theme folder

Copy the obsidian theme folder into the Paymenter themes directory.

/var/www/paymenter/themes/

You should end up with: /var/www/paymenter/themes/obsidian

2) Upload the updated FilamentInput.php

Replace (or upload) the updated FilamentInput.php into the Paymenter app classes directory: 

/var/www/paymenter/app/Classes/

3) Checkout icons setup (SVG)

For checkout icons to work, upload an SVG file into the images directory using the exact product name. Example: for Minecraft Forge, name the file: 

MinecraftForge.svg

Upload it here:

/var/www/paymenter/public/assets/images/

Tip: Keep the filename exact (no spaces). Use PascalCase like MinecraftForge.svg.

4) Build the theme assets

From your Paymenter root directory, run:

npm run build obsidian

If you see an error, make sure Node/NPM are installed and you are in the correct directory.

Section 2 - Install the Extensions

Extensions to upload

Copy these extension folders into:

/var/www/paymenter/extensions/Others/
  • ObsidianFooterEditor
  • ObsidianTermsAndCondition
  • ObsidianAboutPage
  • ObsidianNavbarEditor
1

Go to the Extensions page

Visit: www.theredomain.com/admin/extensions/extension

Replace www.theredomain.com with your real domain.

2

Install / Upload and enable

For each extension, click Ready to Install (or Upload) and then Enable.

3

Configure in the Obsidian Theme section

After enabling them, scroll the left sidebar in admin and you will see an Obsidian Theme section. From there you can edit each extension to match your needs.

Recommended: Clear caches if something does not show

If an extension page or UI does not appear, run these from the Paymenter root:

php artisan view:clear
php artisan cache:clear
php artisan config:clear

Need help?

If you get stuck, join the Discord support: https://discord.gg/hkrxptUjmV

If you are happy with everything after installing and using it, please consider leaving a review.

Done

Once the theme is installed and the extensions are enabled, you are ready to customize your navbar, footer, terms, and about pages from the admin panel.

If anything looks wrong after install, restore from your backup and ask in Discord.
